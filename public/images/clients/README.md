## Logo Specifications

- Format: PNG (with transparency preferred)
- Recommended size: 300x150px or similar aspect ratio
- Background: Transparent or white
- The component will automatically handle grayscale styling and hover effects

If a logo file is missing, the component will display the client name as a fallback.
